You are an expert full-stack developer. Build a complete production-ready B-BBEE (Broad-Based Black Economic Empowerment) compliance web application that matches an Excel toolkit exactly. This is for South African companies.

## PROJECT OVERVIEW
Build a web app where users can:
1. Upload their B-BBEE Excel toolkit
2. See their scorecard on a beautiful dashboard
3. Edit pillar data and see scores update in real-time
4. Create "what-if" scenarios
5. Export updated Excel and PowerPoint reports

## TECH STACK
- **Frontend**: Next.js 14 (App Router), Tailwind CSS, shadcn/ui components
- **Backend**: Next.js API routes (no separate backend needed initially)
- **Database**: PostgreSQL with Prisma ORM (for Phase 2 - start with file-based)
- **Excel Processing**: SheetJS (xlsx) for reading/writing Excel
- **Deployment**: Ready for Vercel
